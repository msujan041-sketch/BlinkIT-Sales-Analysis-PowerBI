# BlinkIT-Sales-Analysis-PowerBI
Interactive Power BI dashboard analyzing BlinkIT grocery sales data. Features data cleaning with Power Query, data modeling, DAX measures, and interactive visuals to track sales, outlet performance, product categories, customer ratings, and KPIs. Built using Excel, Power Query, DAX, and Power BI.
# 1. Project Title
BlinkIT Grocery Sales Analysis Dashboard using Microsoft Power BI
# 2. Project Description
This project demonstrates the application of Business Intelligence techniques using Microsoft Power BI to analyze BlinkIT grocery sales data. The dashboard provides a comprehensive view of sales performance, outlet analysis, customer ratings, and product distribution through interactive visualizations. The raw Excel dataset was transformed using Power Query, modeled into a structured data model, and analyzed using DAX measures to generate business insights. The dashboard enables users to explore data dynamically using filters and slicers, making it easier to identify trends, compare performance, and support data-driven decision-making. The project is designed to showcase practical Power BI development skills for technical interviews and GitHub portfolio presentation.
# 3. Problem Statement
Retail businesses generate large volumes of transactional data every day, making it difficult to monitor performance through traditional reporting methods. Decision-makers require interactive dashboards that provide quick access to key business metrics and performance indicators. 
This project addresses these challenges by using Power BI to:

● Analyze overall sales performance

● Compare outlet performance across different locations and sizes
● Evaluate product category contributions
● Monitor customer ratings and purchasing trends
● Generate interactive business reports for decision-making

8. Key Insights Derived from Analysis
## The dashboard provides several valuable business insights:

● Tier 3 outlet locations contribute the highest overall sales among all outlet locations.
● Medium-sized outlets generate a larger share of revenue compared to small-sized outlets.
● Regular fat-content products account for a significant portion of total sales.
● Fruits & Vegetables and Snack Foods are among the highest-selling product categories.
● Supermarket Type 1 contributes the largest percentage of total sales.
● Customer ratings remain consistently high across most outlet types, indicating positive customer satisfaction.
● Sales trends across establishment years reveal the growth and performance of outlets over time.

These insights demonstrate how Power BI can convert raw business data into meaningful information that supports strategic business decisions.
